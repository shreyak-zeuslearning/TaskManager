# Task Manager Application
A task manager application with options to add , delete and mark as complete options 
## Technology Used
HTML 
JavaScript
CSS(inline)
Data stored in Local browser storage 
## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task
## How to Run
Open the index.html file directly in the browser 
To view the data 
1. Open the dev tool console 
2. Go to applications
3. Local storage 
4. Task are stored there
