# Task Manager Application
A task manager application with options to add , delete and mark as complete options . Also has drag and drop funcitonality for reordering. 

## Technology Used
- HTML 
- JavaScript
- CSS
- Data stored in Local browser storage 

## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task
- Reorder tasks using drag and drop 
- Preserve the updated task order after page refresh using local storage 

## How to Run
Open the index.html file directly in the browser 
To view the data 
1. Open the dev tool console.
2. Go to applications.
3. Local storage.
4. Task are stored there.
